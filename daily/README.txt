Daily WordPress Theme v2.0
http://www.theme-junkie.com/themes/daily


INSTALL: 
1. Upload the theme folder via FTP to your wp-content/themes/ directory.
2. Go to your WordPress dashboard and select Appearance.
3. Select Daily WordPress theme.
4. Inside your WordPress dashboard, go to Appearance > Daily Theme Options & Appearance > Widgets and configure them to your liking.
